# Systèmes d'Automatisation, Workflows & Agents IA (Techie Lab)

## ⚙️ Les Workflows "Produits" (A partager sur The Traveling Techie / A vendre en entreprise)

### 1. "AI-Driven Cybersecurity Compliance Agent"
- **Concept :** Un agent LLM intégré au système GIT de l'entreprise (ex: GitLab, Azure DevOps) qui surveille l'apparition de vulnérabilités (dépendances, secrets en clair) et soumet automatiquement des Pull Requests correctives en appliquant les recommandations Zero Trust.
- **Modèle de distribution :** Code open-source partiel (Hook CLI) + Intégration sur mesure vendue (SaaS/On-premise).
- **Stack / Outils :** Scripts Python/Go, APIs LLM locales (Ollama pour compliance), GitHub Actions / GitLab CI.

### 2. "The CTO Knowledge Graph" (Agentic RAG)
- **Concept :** Un système Retrieval-Augmented Generation qui ingère tous les documents Terraform, Wiki, Jira, Google Docs d'une équipe technique pour répondre aux questions d'architecture en quelques secondes ("Où tourne le service X ? Pourquoi on a choisi GCP pour Y ? Quel impact de couper ce serveur ?").
- **Modèle de distribution :** Prestation de déploiement d'un MCP Server privé (Model Context Protocol) sur-mesure pour un C-Level (DSI, CTO).

### 3. L'Auto-Publishing "Techie Pipeline"
- **Concept :** Un pipeline n8n ou Make.com qui automatise la curation de contenu. Dès que l'utilisateur lit un article tech intéressant (via RSS ou Notion), le pipeline :
    1. Résume le contenu via LLM.
    2. En déduit les insights stratégiques et l'impact Cloud/Geopolitique.
    3. Prépare un brouillon d'article pour le blog `Techie Dispatch`.
    4. Crée une publication LinkedIn optimisée et l'enregistre en brouillon (ex: Typefully).
- **Modèle de distribution :** Partage du blueprint JSON (gratuit) pour fidéliser, ou vendu en tant que module premium "Personal Branding Automation".

### 4. "The FinOps Automator"
- **Concept :** Script de Workflow branché sur les facturations AWS/Azure. Analyse via LLM : "Voici ma facture de 15 000€. L'IA détecte les EC2 non utilisés, les IPs inactives, et envoie un email avec la commande Terraform exacte pour supprimer ces ressources, attendant seulement l'approbation."
- **Modèle de distribution :** Vente du script et de son intégration. Utilisé en consulting FinOps pour un gain de temps massif.
