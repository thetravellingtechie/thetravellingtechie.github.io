# Cours, Vidéos & Formations (Techie Learn)

## 🎓 Parcours Payants / Haute Valeur (Vente B2C / Formation continue)

### 1. Masterclass "Vibe Coding : De Zéro à Produit IA" (B2C & B2B)
- **Cible :** Leaders techniques, Product Managers, Développeurs classiques.
- **Syllabus (Modules) :**
  1. Les bases des LLMs locaux vs API commerciales.
  2. Setup de l'IDE ultime de l'IA (Cursor, Copilot, extensions).
  3. L'art de "l'ingénierie de prompt" pour le développement en temps réel.
  4. Sécuriser et fiabiliser la production générée par IA.
  5. Étude de cas (Création de 0 à 1 d'un SaaS interactif, hébergé dans le cloud gratuit).
- **Format :** Série de courtes vidéos (Loom / Teachable), un guide complet, + une communauté privée Slack/Discord.

### 2. Bootcamp "Cloud Architecture & Zero Trust" (Intensif)
- **Cible :** Entreprises investissant dans la montée en compétences Cloud de leurs sysadmins.
- **Syllabus (Modules) :**
  1. Différences AWS / Azure / GCP. Quel choix d'infra selon le business ?
  2. Infrastructure as Code via Terraform (Modules avancés, CI/CD).
  3. Kubernetes expliqué simplement avec des schémas dessinés.
  4. Réseau & Sécurité: Architecture "Zero Trust" implémentée dans la réalité.
- **Format :** Session Live Cohort-Based (groupes de 10) ou elearning (vidéos + exercices). Vente B2B (prise en compte par budget de formation CPF / OPCO en France, formation entreprise au Canada/Afrique).
